# Bot app

A bot app base on telebot lib.

## Requirements

Python 3.12

## Getting Started

1. ```python -m venv .venv``` (if it does not exist or is incompatible)
2. ```./.venv/Scripts/activate```
3. ```pip install -r .\requirements.txt```
4. *Up your redis*
5. *Configure the .env file*
6. ```cd src```
7. ```py bot.py```
